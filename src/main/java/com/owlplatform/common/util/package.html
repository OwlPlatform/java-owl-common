Common utilities and static classes used within the Owl Platform.

The utilities package contains a number of helpful classes that may be convenient to developers, or are in use by multiple libraries or components of the Owl Platform. Currently, the contents are as follows:

  • HashableByteArray - A simple class that allows byte[] values to be used in hashing data structures.
  • LRUCache - A simple class that provides a basic Least-Recently Used (LRU) cache based on access time.
  • NumericUtils - A static class that provides methods to convert large numeric/binary data (primarily byte[]) into Strings.
  • Pair - A generic pair class.
